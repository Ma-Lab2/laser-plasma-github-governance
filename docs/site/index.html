Core Principles

这套系统真正想解决什么问题

不是为了多一层流程

而是为了避免“只在某个学生电脑里有代码”“毕业后没人知道版本基线”“一堆人同时改 main 导致历史不可追踪”。

不是只有管理员才需要懂规则

普通成员也必须知道什么是 tag、什么是 Base-Version、为什么不能直接 push main。

AI 不是治理豁免通道

无论是 Codex、Claude Code、Cursor 还是 Copilot，最终都必须落回仓库既有脚本、版本规则和人工 review。
